Assumptions
- Java Installed
- Cucumber Installed
- Maven Installed
- Selenium web driver for java is installed
- Browser Used is Google Chrome
	- Chrome Driver is installed
- Eclipse or Any IDE Installed
	- Cucumber Plugin for Eclipse

1) Create a file find_restaurant.feature with the given scenario.

2) Create step definition file in java to define the Scenario with Given, When and Then.

3) To run these files for the website, create corresponding selenium web driver java file.

4) Execute the feature file.
